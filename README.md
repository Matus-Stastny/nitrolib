Use **yarn** as a package manager.

### Scripts

Run `yarn gulp fetch` before developing.

- `yarn start` - Dev server at **:8080**
- `yarn relay` - GraphQL

Static analysis:
- `yarn test`
- `yarn lint`
- `yarn flow`
- `yarn madge`
